# Strength Programme — endurance support (signed off by Jamie 2026-06-10)

Two sessions/week (cap = `strength_max` in day_rules), 35–45 min, RPE-based —
no 1RM testing (consistent with the no-field-tests policy). Loading guidance is
standard concurrent-training practice (Rønnestad & Mujika review lineage);
adjust per athlete feedback.

**Equipment is asked EVERY week** in the Sunday plan message — availability
changes with travel. Sessions are pushed with the Tier C (no-equipment) content
by default and upgraded via edit_workout when the athlete answers.

## Equipment tiers

| Tier | Equipment | Main-lift mapping |
|---|---|---|
| A | full gym | back squat / leg press, barbell RDL, barbell hip thrust |
| B | dumbbells or kettlebells (hotel/home) | goblet squat, single-leg DB RDL, DB split squat, KB swing |
| C | bodyweight + band (default — always possible) | rear-foot-elevated split squat, single-leg RDL (reach), step-ups (stairs/bench), banded hip thrust |

## Session template (A/B alternation across the week)

| Block | Session A | Session B |
|---|---|---|
| Warm-up 5′ | easy spin/row + leg swings | same |
| Main 20′ | squat pattern 3×5 @ RPE 8 + hinge (RDL) 3×6 | split squat / step-up 3×6 per side + hip thrust 3×8 |
| Ankle 8′ | eccentric calf raises 3×12/side · single-leg balance with perturbation · banded eversion | same — permanent block (ankle protocol), all tiers, both sessions |
| Core 7′ | side plank · Pallof press (band) · dead bug — 2 rounds | same |

Tier C reps shift up (3×8–10) where load can't reach RPE 8.

## Phase periodisation

| Phase | Frequency | Emphasis |
|---|---|---|
| Base / Build | 2×/wk | full template, progress load weekly |
| Specific | 2×/wk | maintain load, 2 working sets |
| Peak | 1–2×/wk | maintenance, nothing new |
| Taper | 1×/wk light; none in race week | neuromuscular touch only |

## Scheduling rules

- Slot 1: the Wednesday spare slot (already designated strength-or-cross-training).
- Slot 2: after a swim day (Tue/Thu) — low interference.
- Never the day before the long ride; ≥8 h from any quality bike/run session.
- Strength never displaces a swim/bike/run to hit TSS (existing planner rule).

## Compliance

Watchdog T11: completed strength sessions below target for 2 consecutive weeks
→ coach-visible flag (3-day nag suppression applies, like all triggers).
